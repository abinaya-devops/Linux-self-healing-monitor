# Linux Self-Healing Monitor

A Bash-based Linux monitoring tool that monitors CPU, Memory, Disk, and Service health.

## Features

- CPU Usage Monitoring
- Memory Usage Monitoring
- Disk Usage Monitoring
- Service Health Check
- Health Logging
- Incident Logging

## Technologies Used

- Linux
- Bash Scripting
- Git
- GitHub

## How to Run

chmod +x monitor.sh
./monitor.sh
